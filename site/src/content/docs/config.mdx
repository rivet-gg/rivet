# Config

The Rivet configuration file can use any of the follow names:

| Name         | Format                                                                                              |
| ------------ | --------------------------------------------------------------------------------------------------- |
| `rivet.json` | JSON                                                                                                |
| `rivet.jsonc` | JSON with comments ([JSONC](https://code.visualstudio.com/docs/languages/json#_json-with-comments)) |
